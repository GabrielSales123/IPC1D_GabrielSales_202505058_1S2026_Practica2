## Práctica 2 - Visualizador de Algoritmos de Ordenamiento

## Introducción



El sistema consiste en un visualizador de algoritmos de ordenamiento desarrollado en Java, el cual permite observar de manera gráfica y en tiempo real el funcionamiento de algoritmos como Bubble Sort, Shell Sort y Quick Sort, mostrando estadísticas como comparaciones, intercambios e iteraciones.  

## Cómo ejecutar el programa

1. Clonar o descargar el repositorio:  
   
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Abrir el proyecto en un entorno de desarrollo como:
   
   - NetBeans
   - IntelliJ IDEA
   - Eclipse

3. Asegurarse de tener instalado:
   
   - Java JDK 11 o superior

4. Ejecutar la clase principal:
   
   com.mycompany.algoritmosdeordenamiento.principal.Main

5. Una vez ejecutado, se abrirá la interfaz gráfica desde donde podrás:
   
   - Ingresar datos manualmente o desde archivo
   - Generar números aleatorios
   - Seleccionar el algoritmo de ordenamiento
   - Elegir el orden (ascendente o descendente)
   - Ajustar la velocidad de ejecución
